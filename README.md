# ZeroFriction_KidsLearning
Aplicación móvil premium de desarrollo cognitivo diseñada para niños de 3 años. Este proyecto nace de la frustración de un padre desarrollador ante el estado del mercado actual: aplicaciones infantiles saturadas de anuncios invasivos, mecánicas frenéticas o suscripciones mensuales costosas que rompen el flujo de atención y frustran a los niños.
